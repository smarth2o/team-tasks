## 2. 결정 사항

MVP 구현 전에 답을 정해야 할 핵심 질문 5가지입니다.

| # | 질문 | 영향 범위 |
|---|------|----------|
| D-01 | **일감의 담당자를 나중에 다른 팀원으로 변경(재배정)할 수 있는가?** | AddTaskDialog 수정 vs 별도 편집 UI 필요 여부 |
| D-02 | **담당자 본인도 일감 상태를 변경할 수 있는가, 아니면 팀장만 가능한가?** | 역할(권한) 분리 구현 여부 — MVP 범위에 로그인이 있는지 결정 |
| D-03 | **완료된 일감은 목록에서 숨길 수 있는가, 아니면 항상 함께 표시되는가?** | 완료 항목 필터 토글 UI 추가 여부 |
| D-04 | **마감일이 지난 일감은 시각적으로 강조(예: 빨간색 날짜)되어야 하는가?** | dueDate 비교 로직 및 스타일 처리 필요 여부 |
| D-05 | **팀원을 삭제하면 해당 팀원에게 배정된 일감은 함께 삭제되는가, 아니면 '미배정' 상태로 남는가?** | removeMember 동작 및 고아(orphan) 일감 처리 정책 |
